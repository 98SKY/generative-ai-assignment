# Assignment 3 – Functions

## Objective

Practice creating reusable Python functions using recursion, lambda, map and filter.

## Files

* task1_discount.py → discount calculation
* task2_factorial.py → recursive factorial
* task3_lambda_gst.py → GST using lambda
* task4_map_gst.py → map() usage
* task5_filter_prices.py → filter() usage
* task6_process_prices.py → combined processing
* task7_menu.py → menu driven program

## How to Run

Activate environment:

```
conda activate genai
```

Run any file:

```
python task1_discount.py
```

## Learning Outcome

* Function creation
* Recursion
* Functional programming (lambda/map/filter)
* Simple real-world menu programs
